A história

Calango Virtual - Desenvolvimento de Software começa apenas com um integrante que resolveu expandir seu conhecimento em diversas áreas do desenvolvimento web. Com o sonho de se tornar fullstack e conseguir autonomia para realizar suas viagens procurei me especializar no desenvolvimento de software para realizar trabalhos remotos. Surgindo alguns clientes pelo caminho, seja para criar site estático ou até mesmo E-commerce, senti a necessidade de criar minha Micro Empresa.
